The Mission

Intelligent
Cyber Defense

ThreatSense is a comprehensive web-based platform designed to monitor, detect, and prevent cyber threats using intelligent algorithms and real-time analysis. Built with Flask, MongoDB, and Machine Learning, the engine analyzes network activity to classify risks as normal, suspicious, or high-risk.

🎯 Purpose

  • 🛡️ Real-time monitoring of network activity.
  • 🛡️ AI-powered threat classification.
  • 🛡️ Exportable detection history.
